Fall Detect APP


###Term Project Description

#####1. The main theme of the term project is on Fall Detection and Gesture Recognition.

#####2. Basic Features (75%) of this project should cover the followings:

######a. Fall Detection based on Accelerometer – your application should be able to detect a fall, and should also detect falling down then climbing up and falling down and stay down

######b. Gesture Detection based on Gyroscope & Accelerometer – your application should be able to detect the following activity status of the user: i) Standing; ii) Sitting; and iii) Lying on bed;

#####3. Extra features (25%) of this project is based on the implementation of services or the realization of ideas from your team

##Accelerometer
Add module named ti.coremotion, there is a file Accelerometer.yml to get the data accelerometer. In home.js, the code means get the data and judge the status which is falling down.

This function can start get accelerometer.value. Give a flag to judge it and do other relative functions,such as send email include location and telephone to your friend.

Gyroscope is a device that uses Earth’s gravity to help determine orientation. Its design consists of a freely-rotating disk called a rotor, mounted onto a spinning axis in the center of a larger and more stable wheel. As the axis turns, the rotor remains stationary to indicate the central gravitational pull, and thus which way is “down.”
