Fall Detect APP


###Term Project Description

#####1. The main theme of the term project is on Fall Detection and Gesture Recognition.

#####2. Basic Features (75%) of this project should cover the followings:

######a. Fall Detection based on Accelerometer ‚Äì your application should be able to detect a fall, and should also detect falling down then climbing up and falling down and stay down

######b. Gesture Detection based on Gyroscope & Accelerometer ‚Äì your application should be able to detect the following activity status of the user: i) Standing; ii) Sitting; and iii) Lying on bed;

#####3. Extra features (25%) of this project is based on the implementation of services or the realization of ideas from your team
