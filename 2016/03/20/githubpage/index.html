HEXO+Github搭建个人博客

以前都是在csdn上写技术博客，但是在豪杰妹子的推荐下，用hexo搭建了自己的博客。github pages官网推荐的是Jeklly，用的ruby，不熟悉，所以用了node.js。我都是照着官网上搭建的，文章会具体提出我在搭建中出现的大大小小问题，分享给大家。

Github Pages搭建

官网上有哦，总共也就3步。

1.Create a respository

在Github中创建一个repository，命名格式为 username.github.io:

2.Clone the repository

git clone https://github.com/IMLemon/Lemon.github.io.git

3.Launch automatic page generator

github pages可以自动生成几个主题，如果觉得没必要的，大家可以忽略这步。

DN解析

我们可以用自己的域名去作为我们的网站地址，因为githubpages的地址实在太长了。

1.域名

首先你要有自己的域名，可以去DNSPod或者Godaddy购买，如果有免费也可以。然后呢需要进行地址解析，我用的DNSPod，豪杰妹子好心的给了我一个域名，可以用段时间。
