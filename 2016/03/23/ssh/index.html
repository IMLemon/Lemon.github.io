#Mac系统快速搭建SSH框架

1.搭建准备

Mac系统快很多，不像win需要环境配置。系统安装：

  • 开发环境语言－java

  • IDE－myecilpse

  • Database-MySql

    最好再安装一个MySQLWorkBench,方便查看数据。

  • 下载mysql-jdbc的jar包。

2.创建web项目

打开myeclipse，new一个新的web project，

一路next，最后finish。

3.导入Structs

项目右键MyEclipse->Project Facets[Capabilities]->Install Apache Structs(2.x)Facet

直接next，默认2.1版本，URL pattern选择/*

在默认基础上勾选Spring Plugin包，finish。

4.导入Spring框架

项目右键MyEclipse->Project Facets[Capabilities]->Install Spring Facet,next后选择3.0版本。在默认基础上勾选Spring AOP,Spring Persistence,Spring Web。

5.连接数据库

在MyEclipse面板上找到MyEclipse Hibernate控制中心。

在DB Browser中新建一个Database Driver。本文选择mysql jdbc桥连接，提前下载好jdbc connection的.jar，配置如下：

连接好数据库，导入H框架，项目右键MyEclipse->Project Facets[Capabilities]->Install Hibernate Facet，选择3.3版本。

勾选Enable Hibernate Annotations Support,去掉Create SessionFactory class?。

选择数据源，finish。

6.创建structs.properties文件

项目邮件New->File,选择在src目录下创建文件名为structs.properties文件，Add新name和value。

7.Deploy Tomcat服务器运行

设置部署的服务器。

最后打开浏览器http://localhost:8080/mySSHweb,出现This is my JSP page。成功！
